## Install

Via Composer

``` bash
$ composer require ozh/log
```

## Usage

``` php
use \Ozh\Log\Logger;

require '../vendor/autoload.php';

$logger = new Logger();
$logger->debug('This is a debug message');
```

See `examples/examples.php` for more examples.

This library is fully tested on PHP 5.3 to 7.2 and HHVM.
